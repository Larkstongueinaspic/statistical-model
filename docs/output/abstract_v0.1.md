# 结果摘要-v0.1

## 样本与方法

- 主样本为 BACI HS07 `2008-2024` 年度双边贸易数据中的 `HS6=848620`（半导体器件或集成电路制造设备）。
- 进口国固定为中国，出口国为 `2008-2024` 年间曾向中国出口该产品的全部 `60` 个国家。
- 面板单元为 `出口国 × 年份`，平衡面板共 `1020` 行，正向贸易记录 `585` 行。
- 基准模型为：
  `ln(import_value+1) = β(US × Post2018) + 出口国固定效应 + 年份固定效应 + ε`

## 描述性事实

- 中国该产品总进口额从 `2008` 年的 `23.31` 亿美元上升到 `2024` 年的 `330.97` 亿美元，整体呈显著扩张趋势。
- 中国自美国进口额从 `2008` 年的 `6.48` 亿美元上升到 `2024` 年的 `32.70` 亿美元，绝对额没有出现持续性下降。
- 但美国份额显著下滑：从 `2008` 年的 `27.81%`、`2017` 年的 `27.92%` 下降到 `2024` 年的 `9.88%`，较 `2017` 年下降 `18.05` 个百分点。
- `2024` 年前五来源国份额为：日本 `29.55%`、荷兰 `28.90%`、新加坡 `15.13%`、美国 `9.88%`、韩国 `4.60%`。来源替代主要集中在日本和荷兰等设备强国。
- HHI 从 `2017` 年的 `0.1787` 升至 `2024` 年的 `0.2089`，说明来源结构没有明显分散化，而是向更少数国家重新集中。

## 基准回归

- 基准回归中，`US × Post2018` 的系数为 `0.3182`，`p=0.1593`，方向为正但统计上不显著。
- 若机械地把对数系数转成幅度，这一估计对应约 `37.5%` 的相对增幅，但由于置信区间较宽且跨过零，不能据此做强结论。
- 这一结果表明：在当前单产品、单政策节点、简化 DID 设定下，无法识别出美国在 `2018` 年后相对其他出口国出现显著负向净效应。
- 描述性“美国份额下降”和回归“不显著负向”并不矛盾。前者反映美国在总进口中的占比下滑，后者检验的是美国相对其他出口国的对数进口变化；在本样本中，美国绝对出口额仍在增长，只是增速落后于日本、荷兰、新加坡等替代来源国。

## 最小稳健性

- 使用 `asinh(import_value)` 作为被解释变量后，`US × Post2018` 系数为 `0.2594`，`p=0.2820`，方向不变、仍不显著。
- 剔除 `2020-2021` 年疫情年份后，`US × Post2018` 系数为 `0.1999`，`p=0.4107`，仍不显著。
- 两项稳健性均表明，v0.1 的核心结论比较稳定：美国份额下降是明确事实，但单产品简化 DID 还不足以支持“显著负向因果冲击”的强表述。

## 当前可写进初稿的结论

- 可以稳妥写入的事实结论：
  - `2018` 年后，中国在半导体制造设备 `848620` 上的进口来源结构明显变化，美国份额显著下降。
  - 日本、荷兰、新加坡等来源国的重要性显著上升，替代效应在描述统计上较为清晰。
  - 进口来源没有明显分散化，反而更集中到少数设备强国。
- 当前不宜写得过满的结论：
  - 不能在 v0.1 阶段直接声称“美国出口管制已经显著压低中国自美国进口额”。
  - 更准确的表述应是：描述统计支持“来源替代”，但简化 DID 尚未给出显著的负向净效应证据。
